# WoTKit Nodes for Node-RED

An input and output WoTKit Node for [Node-RED](http://nodered.org).

## Installation

These nodes can be installed using npm.

To install

    cd node-red
    npm install git+ssh://git@github.com:SenseTecnic/wotkit-node.git
